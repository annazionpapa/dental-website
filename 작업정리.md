### 11. 관리자 시스템 구축 - Supabase 연동 (세션 5)

#### 논의 과정
1. **카카오톡 상담**: `href="#"`으로 아무 동작 없음 → 카카오톡 채널 개설 후 링크 연결 필요
2. **온라인 상담 예약**: `alert()`만 뜨고 저장 안 됨 → Supabase DB 저장 방식 채택
3. **알림 방법 비교**:
   - Web3Forms (이메일) / Supabase (DB+관리페이지+알림) / 텔레그램 봇
   - 카카오 알림톡: 사업자등록+심사+유료 → 비현실적
   - 텔레그램 봇: 무료, 무제한, 2분이면 생성 → 향후 고려

## Unsplash 이미지 검증 방법 (향후 참고)
Unsplash photo ID만으로 이미지 내용을 알 수 없으므로:
1. `https://unsplash.com/napi/search/photos?query=검색어&per_page=20` API 사용
2. 응답에서 `urls.regular` 필드의 실제 CDN URL 확인
3. 프리뷰에서 반드시 눈으로 확인
4. `plus.unsplash.com/premium_photo-*` URL은 프리미엄이므로 주의
