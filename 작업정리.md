## 완료된 작업 (세션 1~3)

### 1. 초기 구축
- Next.js 프로젝트 생성, 전체 페이지 구조 완성
- 홈, 진료 안내, 의료진, 병원 소개, 연락처, 치료 사례 등 전체 페이지

### 2. 강남 치과 벤치마킹 기반 디자인 개선
- 히어로 배경: 실제 치과 이미지 + 오버레이 (teal-dark 그라데이션)
- 진료 카드: 이미지 + 아이콘 오버레이 + 텍스트 레이아웃
- 시설 섹션: Unsplash 이미지 카드 형태
- 의사 스니펫: 실제 사진으로 교체
- 모든 페이지 히어로에 배경 이미지 추가

### 3. 히어로 텍스트 가독성 개선
- 오버레이: `from-teal-dark/65 via-teal-dark/50 to-teal-dark/75`
- 텍스트: `font-extrabold`(h1), `font-semibold`(부제) - 그림자 없음
- 여러 시도 끝에 현재 방식으로 확정 (글래스박스 ❌, 텍스트 쉐도우 ❌)

### 4. 헤더 개선
- 메뉴 폰트: `text-sm` → `text-base` (14px → 16px)
- "예약 상담" 버튼: 조건부 스타일링
  - 스크롤 전(투명 헤더): `bg-white text-teal-dark`
  - 스크롤 후(흰 헤더): `bg-teal text-white`

### 5. 진료 갤러리 이미지 추가 (이번 세션)
- `treatments.ts`에 `gallery: GalleryImage[]` 필드 추가
- 각 진료 6개 × 3장 = 18장 이미지 (Unsplash API 검증)
- `treatments/page.tsx`에 3열 이미지 그리드 UI 추가
- 레이아웃: 타이틀 → **갤러리(3장)** → 2컬럼(설명+과정)

### 6. 의료진 실제 사진 반영 (이번 세션)
- `이미지/` 폴더 → `public/images/team/`에 복사
- 4명: 김민수.jpeg, 박서연.jpeg, 이하은.jpeg, 장안나.jpeg
- 최준혁 → **장안나**(여성)로 변경
- 적용 위치: team 페이지, about 원장 인사말, 홈페이지 의료진 스니펫

### 7. 갤러리 이미지 품질 개선 (세션 4)
- 부적합한 이미지 9장 교체 (동남아풍, 흑백, 외국인 위주 → 고급/현대적/아시아 맥락)
- **임플란트**: 3D CT → about 페이지 CT 이미지 재활용, 첨단 장비 → 수술용 현미경
- **심미치료**: 정교한 심미 도구 → LED 의료 장비 (고급스러운 느낌)
- **충치/보존치료**: 최신 치료 장비 → 컬러 진료실 (흑백 → 컬러)
- **소아치과**: 3장 전부 교체 (히잡/외국인 → 아시아 아이 치과 진료 장면)
- **구강외과**: 전문 의료진 케어 → 치과 팀 진료 장면
- **병원 소개**: 구강스캐너(iTero) → 디지털 스캐닝 장비 (노트북 → 실제 장비)

## 현재 알려진 이슈

### 이미지 관련
- Unsplash 이미지는 ID만으로 내용 보장 불가 → API 검증 필수
- 이전에 깨진 이미지 교체 이력:
  - 심미치료 메인: `photo-1581585098574-1c5c4f7f8e26` → `photo-1598256989800-fe5f95da9787`
  - 네비게이션 임플란트: `photo-1581093458791-9f3c3900df4b` → `photo-1551076805-e1869033e561`
- 갤러리 이미지도 1차 시도에서 음식/뇌모델 등 엉뚱한 사진 나와서 Unsplash API(`/napi/search/photos`)로 검증 후 전부 교체

## Unsplash 이미지 검증 방법 (향후 참고)
Unsplash photo ID만으로 이미지 내용을 알 수 없으므로:
1. `https://unsplash.com/napi/search/photos?query=검색어&per_page=20` API 사용
2. 응답에서 `urls.regular` 필드의 실제 CDN URL 확인
3. 프리뷰에서 반드시 눈으로 확인
4. `plus.unsplash.com/premium_photo-*` URL은 프리미엄이므로 주의
